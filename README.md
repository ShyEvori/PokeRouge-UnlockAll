# PokéRogue – Unlock All Script

> ⚠️ **Unofficial / Experimental Project**

## Overview

This repository contains an **experimental browser script** for the web game **PokéRogue**.  
The script modifies the **client-side game state at runtime** to mark all available in-game content as **unlocked**.

This project is intended for **educational, debugging, and experimental purposes only**.

---

## Features

- Client-side manipulation of game state
- Sets internal unlock flags to unlocked
- No server communication or external API usage
- Runs entirely in the browser context
- No persistent modifications (usually resets on reload)

---

## Technical Details

- Language: JavaScript
- Execution context: Web browser (game page)
- Scope: Client-side only
- Dependencies: None

The script does **not** interact with servers, databases, or external services.

---

## Intended Use

This project is aimed at:

- Developers and technically curious users
- Learning about client-side game logic
- Analyzing browser-based game architectures
- Private testing and experimentation

This is **not** an official tool, mod, or extension.
